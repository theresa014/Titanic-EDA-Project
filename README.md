# Titanic-EDA-Project

##  Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand survival patterns.

##  Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

##  Data Cleaning Steps

* Filled missing values in Age using median
* Filled missing values in Embarked using mode
* Dropped Cabin column due to excessive missing values
* Created Age groups using pd.cut()

##  Key Analysis

* Survival rate by Gender
* Survival rate by Passenger Class
* Survival rate by Age Group

##  Visualizations

* Bar plot (Gender vs Survival)
* Bar plot (Class vs Survival)
* Histogram (Age distribution)

##  Key Insights

* Females had higher survival rates than males
* First-class passengers had better survival chances
* Age played a role in survival probability

##  Conclusion

This project demonstrates data cleaning, analysis, and visualization using real-world data.
